# Aman Singh Portfolio Site

This is a static personal website ready to deploy on Vercel.

## Local preview

Open `index.html` in your browser, or run a simple static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Deploy to Vercel with CLI

1. Install Vercel CLI:

```bash
npm i -g vercel
```

2. From this folder, run:

```bash
vercel
```

3. For production deployment:

```bash
vercel --prod
```

## Deploy with GitHub

1. Create a new GitHub repo
2. Upload these files
3. In Vercel, import the GitHub repo
4. Framework preset can stay as "Other" or auto-detected static site
5. Deploy

## Files

- `index.html` — main website
- `styles.css` — site styling
- `Aman_Singh_Resume.pdf` — linked downloadable resume
- `vercel.json` — basic clean URL config
